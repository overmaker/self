时间旅行者


​在我们周围，隐藏着一群时间旅行者

她在世界屋脊，用引力波听宇宙秘密

他触碰恐龙骨架，建造新侏罗纪公园

他深入山林，阅读石头里的地球往事

他寻访丝绸之路国家，记录文明之光

她用镜头捕捉时光，看它残酷的温柔

他抓取社交数据，找到你心里的树洞

六位时间旅行者，他们运用各自技能

解码时间的意义

7月1日 北京·海淀文化馆小剧场

邀你一起穿越时空



  演讲嘉宾：


李虹 中国科学院高能物理研究所

“在世界屋脊倾听宇宙的初啼”

中科院高能物理研究所研究员，全球海拔最高的原初引力波观测站——“阿里计划”执行副总经理。致力于宇宙微波背景辐射极化观测、原初引力波探测、宇宙起源及演化、CMB观测数据分析和参数拟合的研究。2014年入选国家基金委人才计划“优秀青年基金”资助，2015年入选“中科院青年促进会”及“卓越青年科学家”。


王琦 英国诺丁汉大学

“真正的博物馆，应是拥有高情商的故事大王。”

诺丁汉大学建筑与建造环境学院助理教授，博士生导师，致力于现代自然历史类博物馆建筑与观展体验设计。王琦与中科院、中国古动物馆、中国建筑设计研究院、中元国际、伦敦自然历史博物馆、牛津大学自然历史博物馆、英国国家宇航中心、以及诺丁汉自然历史博物馆等科研科普机构保持着长期稳固的合作关系。作为《2017年英国恐龙特展：中国龙——从撼地巨人到飞羽精灵》的策展人之一，获得诺丁汉大学最高学术贡献荣誉——校长勋章。


​

李宏博 中国地质博物馆

“揭秘一桩亿万年前的‘地质大案’”

中国地质博物馆岩石矿物研究室副研究员，致力于岩浆岩岩石学、地球化学、地幔柱与大火成岩省事件的研究以及地学科普工作。发表学术论文三十余篇，科普作品曾获全国国土资源优秀科普作品奖。


​

汪滢滢 独立摄影师

“繁花渐隐，方知人生真义。”

独立摄影师，富士胶片全球签约摄影师，美国艺术摄影中心2017先锋艺术家。《四十/一九七六》、《洄》、《第四行诗》、《黄河以北》等系列摄影作品在平遥国际摄影艺术展、美国艺术摄影中心先锋艺术展、西湖艺术博览会、东京都富士摄影艺术中心等地进行展出。


陈巍 中国科学院自然科学史研究所

“历史思维可以帮助我们把过去的遗迹串珠成线，织线成服。”

中科院自然科学史研究所副研究员，致力于古代科技知识的跨文明比较与传播史研究。对古代丝绸之路沿线国家多次进行实地考察，著有《古代丝绸之路与技术知识传播》一书，以案例的形式，初步探索了科技知识沿着古代丝绸之路在欧亚大陆这一广阔范围内交流传播的历史，另发表相关论文数篇。


朱廷劭 中国科学院心理研究所

“跨越生死时空的心灵对话”

中科院心理所研究员，博士生导师，中国科学院“百人计划”学者。朱廷劭及其团队开展心理与行为科学大数据的交叉研究，实现了对用户心理特征的及时有效的识别，为心理学研究提供了新的思路。先后主持承担国家自然科学基金委面上项目、科技部973和863、中科院A类先导专项等多项研究课题，发表论文60余篇。

SELF格致论道介绍

“SELF格致论道”讲坛是中国科学院全力推出的公益讲坛，致力于精英思想的跨界传播，由中国科学院计算机网络信息中心和中国科学院科学传播局联合主办。SELF是 Science, Education, Life, Future的缩写，旨在以“格物致知”的精神探讨科技、教育、生活、未来的发展。SELF尝试打破过去纯粹以“知识传播”为主的科普形式，专注于思想的传播，力图从思想的源头上促进公众参与科学的积极性，打造具有中国科学院特色的、融合创新、科学与人文关怀并存的公益演讲品牌。
